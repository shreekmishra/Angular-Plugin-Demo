Introduction

The 'ng-Growl-Alert' is an Angular plugin designes to provide informative messages in the browser. This plugin has veriety of API to display alert message, confirm box and growl messages.

Installation

First you need to initialize an Angular project. If you are new to Angular and don't know how to create a Angular project, click here to learn Angular. 

$ ng new demo
$ cd demo

Execute the below command from project directory. This will download the module and add the dependency to your project. 

$ npm install ng-growl-alert

Next, you have to add NgGrowlAlertModule to your root module.

Include Module

You need to include the NgGrowlAlertModule in all modules where you need it. 

import {{'{'}} NgGrowlAlertModule {{'}'}} from 'ng-growl-alert';

@NgModule({{'{'}}
  declarations : [
    ...
  ],
  imports: [
    BrowserModule,
    AppRoutingModule,
    NgGrowlAlertModule
  ],
  providers: [],
  bootstrap: [AppComponent]
{{'}'}})
export class AppModule {{'{ }'}}

Final Step

You have to add the ng-growl-alert component in root component html (As the last line). 

...
<router-outlet></router-outlet>
...
<ng-growl-alert></ng-growl-alert>

Finally, you just need to import one service NgGrowlAlertService from ng-growl-alert to your component, where you want to use. 

import {{'{'}} NgGrowlAlertService {{'}'}} from 'ng-growl-alert'; 
...

@Component({{'{'}}
    selector: 'app-root',
    templateUrl: './app.component.html',
    styleUrls: ['./app.component.css']
  {{'}'}})
  export class AppComponent implements OnInit {{'{'}}
    
    constructor(private alertService : NgGrowlAlertService) {{'{'}}
      
    {{'}'}}
  
    ngOnInit(): void {{'{'}}
    }
  {{'}'}}

With this final step, we are done with the setup. Now, let's move to first demo.
