Alert Box

Alert Box is a special dialog box that inform users about some specific task or it may contain some critical information. ng-growl-alert provides the ability to create an alert box dinamically.Here, user can acknowledge the information by clicking the okay button. 

this.alertService.alert("Get ready to learn.");

How to use

To create an alert box, first, we need to import NgGrowlAlertService service and inject in our component through constructor that we did in last chapter. Next, we can use this services to create an alert box as shown in example. 

this.alertService.alert("My first alet box complete.");

Parameters

The function alert of NgGrowlAlertService accepts few parameters to customize the visuals and the displayed message.

Name Type Default Description
message String The message to be shown in the alert box.
title String null This will be shown as the title of alert box. By default there will not be any title.
allowClose boolean false If true, user can close the alet box by clicking the backdrop or the cross button and also using escape button. 


this.alertService.alert("The only message alert.");
this.alertService.alert("The alert with message and title.", "Alert Message");
this.alertService.alert("The alert with message and title.", "Alert Message", true);

Response Handling

The alert function returns an Observable that you can subscribe to get the user's reponse against the alert box. The function you pass to subscribe, should accept one String value or an object of AlertConfirmResponse Enum.
The Enum AlertConfirmResponse has two propeties for alert box i.e., ALERT_OKAY and ALERT_CLOSE. The value ALERT_OKAY indicates that user clicked 'Okay' button and ALERT_CLOSE indicates that user closed the alert box by clicking the cross button or backdrop or by pressing escape button. 

this.alertService.alert('This example to demontrate the return value', 
    'Alert Response Example', true)
    .subscribe( value => {{'{'}}
        if(value == AlertConfirmResponse.ALERT_OKAY) {{'{'}}
            alert('User clicked \'Okay\' button');
        {{'}'}} else if(value == AlertConfirmResponse.ALERT_CLOSE) {{'{'}}
            alert('User closed the alert box');
        {{'}'}}
    {{'}'}});
